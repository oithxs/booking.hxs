# リリースノート v1.1.0
**リリース日:** 2025年11月12日

## 📢 ハイライト

予約作成時に日付・時刻の候補が自動表示されるオートコンプリート機能を追加！

## ✨ 新機能

### 日付・時刻のオートコンプリート

予約作成（`/reserve`）コマンドで、日付と時刻の入力時に候補が自動表示されるようになりました。

#### 日付入力の補完

入力フィールドをクリックすると、以下の候補が表示されます：

- **今日** - 今日の日付（例: 2025-11-10）
- **明日** - 明日の日付（例: 2025-11-11）
- **明後日** - 明後日の日付（例: 2025-11-12）
- **1週間後** - 7日後の日付（例: 2025-11-17）
- **2週間後** - 14日後の日付（例: 2025-11-24）
- **任意入力** - 任意の日付（例: 2025-11-28）

文字または日付を入力すると、候補が自動的に絞り込まれます。

#### 時刻入力の補完

**開始時刻**: 09:00 〜 21:00 まで、30分刻みで候補が表示されます。

**終了時刻**: 開始時刻が選択されている場合、開始時刻より後の時刻のみが表示されます。
## 🚀 改善・変更
### スマート日時入力
桁数に関係なく入力することができるようになりました
#### 従来
- 2025/1/5 → 2025/01/05の入力が必須
- 9:00 → 09:00の入力が必須
#### 現在
- 2025/1/5 → 受付可能
- 9:00 → 受付可能
