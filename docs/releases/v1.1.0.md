# v1.1.0 - オートコンプリート機能

**リリース日**: 2025-11-12<br>
**種別**: マイナーリリース

## ✨ 新機能

### 📝 日付・時刻のオートコンプリート

予約作成（`/reserve`）コマンドで、日付と時刻の入力時に候補が自動表示されるようになりました！


## 🎯 主な改善点

### 日付入力の補完

入力フィールドをクリックすると、以下の候補が表示されます：

- **今日** - 今日の日付（例: 2025-11-10）
- **明日** - 明日の日付（例: 2025-11-11）
- **明後日** - 明後日の日付（例: 2025-11-12）
- **1週間後** - 7日後の日付（例: 2025-11-17）
- **2週間後** - 14日後の日付（例: 2025-11-24）
- **1ヶ月後** - 30日後の日付（例: 2025-12-10）

#### 入力中の絞り込み

文字を入力すると、候補が自動的に絞り込まれます。

**例**:
- 「2025-11」と入力 → 11月の日付のみ表示
- 「今日」と入力 → 今日の日付が表示

---

### 時刻入力の補完

#### 開始時刻（start_time）

09:00 〜 21:00 まで、30分刻みで候補が表示されます。

```
09:00, 09:30, 10:00, 10:30, 11:00, 11:30, ...
```

#### 終了時刻（end_time）

開始時刻が選択されている場合、**開始時刻より後の時刻のみ**が表示されます。

**例**:
- 開始時刻が 14:00 の場合
- 終了時刻の候補: 14:30, 15:00, 15:30, ... 21:00, 21:30

これにより、**開始時刻より前の終了時刻を選べない**ようになり、入力ミスを防げます。


## 💡 使用例

### `/reserve` コマンドでの使用

1. `/reserve` と入力
2. `date` フィールドをクリック
3. 「今日」「明日」などの候補から選択、または日付を入力
4. `start_time` フィールドで時刻を選択
5. `end_time` フィールドで開始時刻より後の時刻を選択

### スマート日時入力との組み合わせ

従来の「スマート日時入力」機能も継続して利用できます。

- `2025/1/5` → 自動的に `2025/01/05` に正規化
- `9:00` → 自動的に `09:00` に正規化
