# リリースノート v1.3.1
**リリース日:** 2025年11月12日

## 📢 ハイライト

予約一覧が複数メッセージに自動分割され、統一フッターが追加されました！

## ✨ 新機能

### 複数メッセージによるページネーション

予約が10件以上ある場合、自動的に複数のメッセージに分割して表示されるようになりました。

**分割ルール**:
- **1件目のメッセージ**: ヘッダー + 最初の9件
- **2件目以降**: 各メッセージに10件ずつ

すべてのページは実行者のみに表示されます（Ephemeral）。

## 🚀 改善・変更

### フッターの統一

すべての埋め込みメッセージに統一フッターが追加されました。

**基本形式**: `部室予約システム | コマンド名`

**進捗付き（リスト表示時）**: `部室予約システム | list | 予約 3/25`

#### メリット

- すべての埋め込みに統一フッター
- どのコマンドで作成されたメッセージか一目で分かる
- 予約一覧の進捗が分かる

## 🐛 バグ修正
なし

## ⚠️ 注意事項 / 既知の問題
- 非常に多くの予約（100件以上）がある場合、メッセージ数が多くなる可能性
