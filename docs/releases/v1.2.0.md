# リリースノート v1.2.0
**リリース日:** 2025年11月12日

## 📢 ハイライト

予約内容を後から編集できる `/edit` コマンドを追加！

## ✨ 新機能

### `/edit` コマンド - 予約編集機能

予約を作成した後でも、日付・時刻・コメントを自由に編集できるようになりました。

#### 編集可能な項目

- **日付** - 予約の日付を変更
- **開始時刻** - 開始時刻を変更
- **終了時刻** - 終了時刻を変更
- **コメント** - コメントを追加・変更・削除

#### 使用例

```
/edit reservation_id:abc123def456 start_time:15:00 end_time:16:00
```

変更したい項目だけを指定すればOK。他の項目はそのまま維持されます。

#### 安全機能

- **重複チェック**: 編集後の日時が他の予約と重複する場合はエラー
- **過去日時のブロック**: 過去の日時には編集不可
- **オートコンプリート対応**: 日付・時刻入力時に候補表示
- **編集可能ユーザ**: 予約IDを知っている人であれば編集可能ですが、予約IDは推測困難であるため安全です。
