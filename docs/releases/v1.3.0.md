### カラーコードの統一

各コマンドに適切な色が設定されました。

| コマンド | 色 | 意味 |
|---------|---|------|
| `/reserve` | 🟢 緑 | 成功・作成 |
| `/edit` | 🟡 黄  | 変更・編集 |
| `/cancel` | 🔴 赤  | キャンセル |
| `/complete` | 🔵 青 | 完了 |
| `/list` | ⚫ 黒  | 情報表示 |
| `/my-reservations` | ⚪ 白 |  情報表示 |
| `/feedback` | 🟢 緑 |  フィードバック |
| エラー | 🔴 赤 |  エラー |

#### メリット

- 色分けで一目で状態を判断
- 絵文字で情報の種類を識別
- 構造化されたレイアウトで読みやすく
- すべてのコマンドで一貫したデザイン

## 🐛 バグ修正

## ⚠️ 注意事項 / 既知の問題
- Discord の埋め込み制限により、1メッセージあたりの情報量に上限があります
- 非常に多くの予約がある場合、複数メッセージに分割される可能性があります
