# v1.3.0 - UI改善（埋め込みメッセージ化）

**リリース日**: 2025-11-12<br>
**種別**: マイナーリリース


## ✨ 新機能

### 🎨 埋め込みメッセージへの全面移行

help以外のすべてのコマンドの応答が、見やすく美しい**埋め込み形式**（Discord Embed）に変更されました！


## 🎯 主な改善点

### 視覚的な改善

#### After（埋め込み形式）


### カラーコードの統一

各コマンドに適切な色が設定されました。

| コマンド | 色 | 意味 |
|---------|---|------|
| `/reserve` | 🟢 緑 | 成功・作成 |
| `/edit` | 🟡 黄  | 変更・編集 |
| `/cancel` | 🔴 赤  | キャンセル |
| `/complete` | 🔵 青 | 完了 |
| `/list` | ⚫ 黒  | 情報表示 |
| `/my-reservations` | ⚪ 白 |  情報表示 |
| `/help` |  　なし | ヘルプ |
| `/feedback` | 🟢 緑 |  フィードバック |
| エラー | 🔴 赤 |  エラー |


## 🎯 ユーザー体験の向上

### 情報の視認性向上

- ✅ 色分けで一目で状態を判断
- ✅ 絵文字で情報の種類を識別
- ✅ 構造化されたレイアウトで読みやすく

### 一貫性のある体験

- ✅ すべてのコマンドで同じデザイン言語
- ✅ 予測可能なUI
- ✅ プロフェッショナルな見た目


## 📝 その他の変更

- `commands/response_helpers.go` - 応答ヘルパー関数を追加
- すべてのコマンドハンドラーを埋め込み形式に変更
- カラーコードの定数化


## ⚠️ 注意事項

- Discord の埋め込み制限により、1メッセージあたりの情報量に上限があります
- 非常に多くの予約がある場合、複数メッセージに分割される可能性があります（次のバージョンで改善予定）
