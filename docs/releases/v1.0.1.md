# リリースノート v1.0.1
**リリース日:** 2025年11月12日

## 📢 ハイライト

自動クリーンアップ処理が毎日決まった時刻に実行されるように改善されました。

## ✨ 新機能
なし
## 🚀 改善・変更

### 自動処理タイミングの最適化

**変更前**:
- Bot起動時刻に依存して実行
- 起動時刻によって実行タイミングがバラバラ

**変更後**:
- **毎日午前3時00分**: 期限切れ予約を自動完了
- **毎日午前3時10分**: 古い予約データを自動削除

### 起動時の即時クリーンアップ

Bot起動が深夜0時〜0時5分の場合、次の実行を待たずに即座にクリーンアップを実行します。

## 🐛 バグ修正

- 起動時のクリーンアップ判定処理で発生していた無限ループを修正

## ⚠️ 注意事項 / 既知の問題
- 午前3時にサーバー負荷が若干上がる可能性があります（通常は軽微）
