# リリースノート v1.0.0
**リリース日:** 2025年11月11日

## 📢 ハイライト

部室予約システムの正式リリース！Discord上で簡単に予約管理ができるようになりました。

## ✨ 新機能

### 📅 予約管理

#### `/reserve` - 予約作成
- 日付、開始時刻、終了時刻を指定して予約を作成
- 予約IDが自動生成され、作成者のみに通知
- 時間の重複を自動チェック
- コメント付きで予約可能

#### `/cancel` - 予約キャンセル
- 予約IDを指定して予約を取り消し
- キャンセル理由をコメントで記録可能

#### `/complete` - 予約完了
- 予約IDを指定して予約を完了状態に変更
- 完了コメントを追加可能

### 📋 予約確認

#### `/list` - 全予約表示
- すべての予約を一覧表示
- 実行者のみに表示

#### `/my-reservations` - 自分の予約表示
- 自分が作成した予約のみを表示
- 実行者のみに表示

### 💬 サポート機能

#### `/help` - ヘルプ表示
- すべてのコマンドの使い方を確認

#### `/feedback` - フィードバック送信
- 意見や要望を**完全匿名**で送信
- システム改善に活用

### 🤖 自動処理機能

- **期限切れ予約の自動完了**: 終了時刻が過ぎた予約を自動的に完了状態に変更
- **古いデータの自動削除**: 30日以上前の完了・キャンセル済み予約を自動削除
- **自動保存**: 5分ごとに予約データを自動保存
### 🛡️ セキュリティ・プライバシー
- 予約IDは予測困難なランダム文字列
- 予約IDは作成者のみに通知
