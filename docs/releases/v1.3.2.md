# リリースノート v1.3.2
**リリース日:** 2025年11月17日

## 📢 ハイライト

セキュリティ強化：予約編集時の公開メッセージから予約IDが除外されました。

## ✨ 新機能
なし

## 🚀 改善・変更
なし
## 🐛 バグ修正

### 予約IDの公開問題を修正

`/edit` コマンド実行時、チャンネルへの公開メッセージに予約IDが表示されていた問題を修正しました。

**修正前**:
- 公開メッセージに予約IDが含まれる
- 誰でも予約IDを確認できてしまう

**修正後**:
- 公開メッセージから予約IDを除外
- 予約IDは実行者のみに表示（Ephemeralメッセージ）

#### セキュリティへの影響

予約IDは予約の編集・キャンセル・完了に必要な重要な情報です。この修正により、予約IDが実行者以外に見えなくなり、セキュリティが向上しました。

## ⚠️ 注意事項 / 既知の問題

- この修正により、ユーザー体験への影響はありません。セキュリティが向上しただけです。
