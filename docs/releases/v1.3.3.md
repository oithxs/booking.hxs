# リリースノート v1.3.3
**リリース日:** 2025年11月17日

## 📢 ハイライト

日付入力のオートコンプリート機能が**曜日表示対応**になりました。予約したい日が何曜日かが一目でわかるようになり、より直感的に日付を選択できます。


## 🚀 改善・変更

### 📅 日付オートコンプリートに曜日表示を追加

日付候補に曜日が表示されるようになりました。

**変更前:**
```
今日
明日
明後日
2025/11/20
2025/11/21
1週間後
```

**変更後:**
```
今日 2025/11/17 (日)
明日 2025/11/18 (月)
明後日 2025/11/19 (火)
2025/11/20 (水)
2025/11/21 (木)
1週間後 2025/11/24 (日)
```

**メリット:**
- 週末（土日）が一目でわかる
- 希望する曜日での予約が簡単
- より直感的な日付選択が可能

**適用範囲:**
- `/reserve` コマンドの日付入力
- `/edit` コマンドの日付変更
- その他すべての日付入力オートコンプリート


## ⚠️ 注意事項 / 既知の問題

特になし。既存の機能に影響はありません。
